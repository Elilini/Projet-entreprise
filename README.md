Pour que votre fichier README.md soit conforme aux exigences de votre projet et évite que votre code ne soit ignoré, il doit être structuré de manière professionnelle et contenir des instructions précises.

Voici le modèle exact à copier et à adapter dans votre fichier :

Projet Algorithmique L2 - Entreprise
Membres du groupe :

[Ton Prénom] [Ton Nom] - Numéro étudiant : [Ton Numéro]

[Prénom du Binôme] [Nom du Binôme] - Numéro étudiant : [Son Numéro]

1. Structure du Dépôt
src/ : Contient l'intégralité du code source (Backend et Frontend) sans aucune archive compressée.

rapport/ : Contient les fichiers sources LaTeX (.tex), le template (.cls) et les images nécessaires à la compilation du rapport.

data/ : Contient les jeux de données utilisés par le programme.

2. Instructions pour exécuter le code
Backend (Logique)
Ouvrez un terminal et accédez au dossier : cd src/backend

Compilez le code : gcc *.c -o projet

Lancez le programme : ./projet

Frontend (Interface)
Accédez au dossier : cd src/frontend

[Ajoute ici la commande pour lancer ton interface, ex: python main.py ou open index.html]

3. Rapport
Le dossier /rapport contient les fichiers sources permettant de compiler le rapport final. Le lien de ce dépôt GitHub est également présent sur la page de garde du rapport PDF.
